# geoffreymugford.com

Personal site for Geoffrey Mugford - freelance game development for education, research and culture. Currently a single freelance landing page, structured to grow into a fuller portfolio (teaching, writing, games, film).

## Stack

Plain static HTML + CSS. No build step, no dependencies (one Google Font, with a serif fallback). Hosted on GitHub Pages.

## Local preview

It's static - open `index.html` in a browser, or serve the folder with any static server.

## Structure

- `index.html` - the page
- `styles.css` - all styles (theme variables at the top)
- `media/` - per-project gameplay video (`*.mp4`) and poster still (`*.jpg`); the ViewfindR poster doubles as the social-share image
